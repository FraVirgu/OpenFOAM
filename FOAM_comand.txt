==================================================
   RUNNING THE icoFoam TUTORIAL (CAVITY CASE)
==================================================

1️⃣  Compile or check icoFoam executable
----------------------------------------
If you want to look at the solver source or ensure it’s available:
    code $FOAM_SOLVERS/incompressible/icoFoam

This opens the icoFoam solver directory in VS Code.

==================================================
2️⃣  Set up the case
--------------------------------------------------
Navigate to your run directory (usually $FOAM_RUN):

    cd $FOAM_RUN

Copy the standard tutorial case:

    cp -r $FOAM_TUTORIALS/incompressible/icoFoam/cavity ./cavity

Enter the new case folder:

    cd cavity

==================================================
3️⃣  Run the mesh generation
--------------------------------------------------
Create the computational mesh defined in system/blockMeshDict:

    blockMesh

This reads `system/blockMeshDict` and generates the mesh files
in the `constant/polyMesh/` directory.

==================================================
4️⃣  Run the simulation
--------------------------------------------------
Execute the solver:

    icoFoam

This runs the transient incompressible laminar flow simulation
inside a square cavity.

During the run, time directories (e.g. 0.1, 0.2, 0.3, …)
are created, each containing the field results (U, p, etc.).

==================================================
5️⃣  Visualize results in ParaView
--------------------------------------------------
You can view both the mesh and the simulation results
using ParaView.

➡ To view only the mesh:
--------------------------------
1. Inside the case folder, create a file named `paraview.foam`  
   (the content can be empty — it just triggers ParaView’s OpenFOAM reader).

2. Run the mesh generator if not done already:
       blockMesh

3. Launch ParaView and open the file:
       paraview paraview.foam

Then click “Apply” to visualize the mesh.

➡ To view the simulation results:
--------------------------------
1. Run the solver:
       icoFoam

2. Open the same file in ParaView:
       paraview paraview.foam

3. Click “Apply” → you can now use the time slider to see
   the field evolution (velocity, pressure).

==================================================
6️⃣  Folder structure overview
--------------------------------------------------
Each OpenFOAM case follows this standard directory layout:

📂 **0/**
   - Contains *initial conditions* for each field solved by the solver.
   - For icoFoam, typical files are:
     - `U` → velocity field
     - `p` → pressure field

📂 **constant/**
   - Contains *mesh* and *physical properties*.
   - Common subfolders:
     - `polyMesh/` → generated by blockMesh
     - `transportProperties` → defines viscosity (ν)
   - Holds parameters that remain constant during the run.

📂 **system/**
   - Defines the *simulation control and numerical setup*:
     - `controlDict` → start/end time, solver name, output intervals
     - `fvSchemes` → spatial/temporal discretization schemes
     - `fvSolution` → linear solver and tolerance settings

==================================================
7️⃣  Quick reference
--------------------------------------------------
| Command        | Purpose                                  |
|----------------|-------------------------------------------|
| blockMesh      | Build the mesh                            |
| icoFoam        | Run the incompressible flow simulation    |
| paraFoam       | Launch ParaView with the current case     |
| paraview .foam | Manually open case in ParaView            |

==================================================
Summary
--------------------------------------------------
- The *icoFoam* solver simulates transient, laminar, incompressible flow.
- The *cavity* tutorial models flow driven by the top moving wall.
- Case setup always follows the 0 / constant / system structure.
- Results can be visualized at any stage with ParaView.
